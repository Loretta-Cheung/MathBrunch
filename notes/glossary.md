# Glossary

## Adjacency matrix

A matrix that records which vertices of a graph are connected by edges.

## Eigenvalue

A scalar that describes how a matrix stretches or compresses a vector direction.

## Eigenspace

The vector space of all eigenvectors associated with a given eigenvalue, plus the zero vector.

## Gram matrix

A matrix of pairwise inner products. For vectors `x_1, ..., x_n`, its entries are `G_ij = x_i · x_j`.

## Spherical code

A finite set of points on a unit sphere.

## Inner product set

The set of all inner products between distinct points of a spherical code.

## T-avoiding code

A spherical code whose inner product set avoids a specified forbidden set `T`.

## Strongly regular graph

A regular graph with uniform numbers of common neighbors for adjacent and non-adjacent vertex pairs.

## Spectral embedding

An embedding of graph vertices into a vector space using eigenvectors or eigenspaces of a graph-related matrix.

## Linear programming bound

A proof method using polynomial certificates and positivity to bound the size or energy of a configuration.

## Universal optimality

A configuration is universally optimal if it minimizes energy for a broad class of potential functions.
