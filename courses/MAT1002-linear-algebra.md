# MAT1002 — Linear Algebra

## Role in the roadmap

Linear algebra is the direct bridge from graph theory to spherical codes.

## Core concepts

- vectors and vector spaces
- inner products
- orthogonality
- projections
- matrices
- eigenvalues and eigenvectors
- diagonalization
- symmetric matrices
- positive semidefinite matrices
- Gram matrices

## Research-facing translation

| Course concept | Research-facing concept |
|---|---|
| inner product | angle and distance on the sphere |
| symmetric matrix | spectral theorem, real eigenspaces |
| adjacency matrix | graph structure encoded as linear algebra |
| eigenspace | geometric embedding space |
| Gram matrix | pairwise inner products of point configurations |
| positive semidefinite matrix | feasibility of an inner-product pattern |

## Study target

The main goal is to understand how matrices create geometry.

## Practice tasks

- Compute eigenvalues of a small adjacency matrix.
- Construct a simple spectral embedding.
- Compute a Gram matrix from point coordinates.
- Check whether a given matrix is positive semidefinite.
- Explain why symmetric matrices have orthogonal eigenspaces.
