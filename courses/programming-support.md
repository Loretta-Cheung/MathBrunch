# Programming Support

## Role in the roadmap

Programming is a tool for mathematical exploration, not only software development.

## Useful tasks

- build adjacency matrices
- compute eigenvalues and eigenvectors
- construct spectral embeddings
- calculate pairwise inner products
- test T-avoiding conditions
- compute sample energy values
- visualize small configurations

## Recommended tools

- Python
- NumPy
- NetworkX
- Matplotlib
- Jupyter notebooks

## Project habit

Every computational notebook should include:

1. mathematical question
2. input object
3. computation
4. output table or plot
5. interpretation
6. next question
