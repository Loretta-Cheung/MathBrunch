# Privacy Policy for Public Notes

This repository is designed for public GitHub use.

## Do not include

- real names of professors, mentors, classmates, friends, or private individuals
- private emails
- private screenshots
- private institutions when they identify a personal relationship
- private messages
- personal medical, legal, immigration, or relationship details
- unpublished private advice from identifiable people

## Use instead

- research mentor
- target research area
- recent reading
- course instructor
- faculty contact
- public paper topic
- learning conversation
- research-facing bridge

## Before pushing

Run:

```bash
python scripts/privacy_check.py
```

Then manually read the diff.

Automation helps, but privacy is a judgment call.
